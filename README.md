# KOZAINEK

Компьютерное зрение для помощи незрячим и слабовидящим людям.

**Цель:** Создать систему распознавания объектов в реальном времени с голосовым выводом на русском языке, которая помогает безопасно ориентироваться в пространстве.

---

## 1. Функциональность (Текущее состояние - MVP+)

На данный момент реализован расширенный MVP со следующими возможностями:

### 1.1. Обнаружение объектов
*   Используется нейросеть **YOLOv8** (модель `yolov8n-seg.pt`).
*   Распознает **80+ типов объектов**: люди, транспорт, животные, мебель и предметы быта.
*   Работает в реальном времени на CPU и GPU (поддержка NVIDIA CUDA).

### 1.2. Анализ расстояния
*   Оценка расстояния до объектов на основе их размера в кадре.
*   Классификация: "близко", "на среднем расстоянии", "далеко".
*   Используются эталонные размеры для 30+ типов объектов.

### 1.3. Определение позиции
*   Определение положения объекта относительно пользователя:
    *   По горизонтали: "слева", "по центру", "справа".
    *   По вертикали: "сверху", "посередине", "снизу".

### 1.4. Обнаружение препятствий
*   Детекция вертикальных линий (углы стен, двери, столбы) с помощью алгоритмов Canny и Hough Transform.
*   Предупреждение: "Внимание! Возможное препятствие впереди".

### 1.5. Голосовой вывод
*   Синтез речи на русском языке (используется системный голос Windows).
*   Асинхронная работа (не блокирует видеопоток).
*   Озвучивание типа объекта, его позиции и расстояния.

### 1.6. Система приоритетов
*   **Высокий приоритет (опасно):** люди, транспорт, собаки, светофоры.
*   **Средний приоритет (препятствия):** стулья, столы, скамейки.
*   **Низкий приоритет:** остальные объекты.

### 1.7. Менеджер объявлений
*   Защита от спама: один и тот же объект не озвучивается чаще чем раз в 4-6 секунд.
*   Автоматическое озвучивание каждые 6 секунд (режим для незрячих).

### 1.8. Визуализация (для зрячих ассистентов)
*   Отображение видео с обводкой объектов:
    *   Красный = близко (опасно).
    *   Желтый = средняя дистанция.
    *   Зеленый = далеко.
*   Отображение уверенности распознавания и сегментационных масок.

## 2. Установка и Запуск

### Требования
*   Windows 10/11 (64-bit)
*   Python 3.10+
*   Веб-камера

### Инструкция
  [скачайте](https://github.com/nakhraddd/kozainekback/releases/)

  Разархивируйте содержимое

  **Запустите kozainek.exe:**
    *При первом запуске приложение автоматически скачает необходимую модель нейросети (`yolov8n-seg.pt`).*

## 3. Использование

### Для незрячего пользователя
1.  Запустите программу.
2.  Направьте камеру перед собой.

### Для ассистента / Разработчика
*   Локальный сервер доступен по адресу: `http://localhost:8000`
*   Список камер: `http://localhost:8000/cameras`
*   Настройки находятся в файле `app/config.py`.
